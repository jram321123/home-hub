# Hogan Home Hub v2.0

Premium AMOLED-friendly dashboard prototype.

## Cards
- Full-width weekly calendar
- Today's agenda
- Google Keep shared list launcher
- Ring app launcher/status card
- SmartThings app launcher/status card
- Google Home app launcher/status card

## Editing
Tap Edit on cards to change placeholder/status text.  
For Shared Lists, use this format:

emoji | list title | Google Keep link

Example:

🛒 | Grocery List | https://keep.google.com/


## Live Calendar
Tap the gear and paste your Google Calendar **Secret address in iCal format** into the Google Calendar iCal field.

This keeps the URL stored only in the tablet browser/local storage, not in the GitHub files.
